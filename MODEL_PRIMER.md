# Comprehensive LLM Guide

## Task Types
1. **Text Generation**
   - Generate creative text based on prompts.
   - Example: Writing stories, poetry, etc.

2. **Text Summarization**
   - Condense longer texts into shorter summaries.
   - Example: Summarizing articles or reports.

3. **Question Answering**
   - Answer questions based on provided context or general knowledge.
   - Example: FAQ sections, customer support.

4. **Language Translation**
   - Translate text from one language to another.
   - Example: English to Spanish.

5. **Sentiment Analysis**
   - Determine the sentiment (positive, negative, neutral) expressed in a text.
   - Example: Analyzing reviews or social media posts.

## Critical Rules
- **Prompt Clarity**: Be clear and concise in your prompts to avoid ambiguity.
- **Context Provision**: Provide necessary context for more accurate results. 
- **Feedback Loop**: Regularly refine prompts and revisit outputs for better performance.
- **Ethical Considerations**: Ensure the content generated adheres to ethical guidelines and avoids harmful biases.

## Quick Reference
- Use clear and precise language.  
- Specify the output format if necessary (e.g. bullet points, paragraphs).
- Review results critically; the model may not always provide correct information.

## Common Mistakes
1. **Vague Prompts**: Using non-specific language that confuses the model.
   - *Example*: "Tell me something interesting."
   - *Better*: "Tell me a surprising fact about marine biology."

2. **Ignoring Output Quality**: Not reviewing the output before use.
   - Outputs should be double-checked for accuracy and relevance.

3. **Poor Context**: Failing to provide context for specialized information.
   - When asking for technical details, include relevant background information.

4. **Overloading Prompts**: Asking too many questions in a single prompt.
   - Break down tasks into manageable parts for better results.
